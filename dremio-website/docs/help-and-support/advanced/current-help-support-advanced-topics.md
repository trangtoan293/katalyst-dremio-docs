# Advanced Topics
This section contains a set of topics about advanced usage scenarios with Dremio:
  * [**Configuration Files**](/help-support/advanced-topics/configuration-files) - Reference information about the `dremio.conf` and `dremio.env` configuration files.
  * **[Connect to a Ranger Host use SSL](/help-support/advanced-topics/hive-ranger-ssl)** - Connect a Ranger host using Secure Socket Layer (SSL) in Dremio.
  * [**Creating and Querying Wide Tables**](/help-support/advanced-topics/wide-tables) - Create wide tables in Dremio and make them available for queries.
  * **[Dremio Preferences](/help-support/advanced-topics/dremio-preferences)** - Customize the behavior of specific features in the Dremio console.
  * [**Gandiva-based Execution**](/help-support/advanced-topics/gandiva) - Learn about Gandiva and supported functions.
  * [**Hash Aggregation Spilling**](/help-support/advanced-topics/hash-agg-spilling) - Learn about hash aggregation spilling implemented in Dremio, how to enable/disable it, and the limitations.
  * **[Query Relational Database Sources Directly](/help-support/advanced-topics/external-queries)** - Run external queries against relational-database sources.
  * **[Ranger-Based Authorization](/help-support/advanced-topics/hive-ranger)** - Check the Apache Ranger policy permissions and then allow/disallow access as defined by the Ranger policy.
  * **[Refresh Metadata](/help-support/advanced-topics/metadata-caching)** - Learn more about automatic refreshes, scheduling refreshes, triggering refreshes, and limitations in how metadata is refreshed.
  * [**Runtime Filtering**](/help-support/advanced-topics/runtime-filtering) - Improve query performance with runtime filtering.
  * **[S3 on Amazon EMR Configuration](/help-support/advanced-topics/hive-s3)** - Configure for S3-backed Hive tables on Amazon EMR.
  * [**Start, Stop, and Status**](/help-support/advanced-topics/start-stop) - Learn how to start up and shut down Dremio on YARN and non-YARN deployments for RPM and Tarball installations.
  * **[Upgrade with Custom Sources](/help-support/advanced-topics/custom-arp)** - Follow steps to safely enable functionality of new features when you upgrade Dremio if your environment utilizes external query (which is only supported via ARP sources).

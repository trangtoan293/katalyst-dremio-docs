## Tags[​](/data-products/govern/wikis-tags#tags "Direct link to Tags")
Tags for datasets offer a powerful way to organize and retrieve datasets within a data catalog. By creating and assigning tags to datasets, users can easily search and filter through large collections related datasets. Tags also enhance the search experience, allowing users to quickly locate datasets associated with a specific tag. By clicking on a tag, users can initiate a search that brings up all datasets linked to that tag, streamlining the process of finding relevant data and improving overall data management.

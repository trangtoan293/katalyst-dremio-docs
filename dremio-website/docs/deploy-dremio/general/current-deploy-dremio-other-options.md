# Other Deployment Options
Besides the [Kubernetes deployment](/deploy-dremio/deploy-on-kubernetes), there are other alternative supported options for deploying Dremio:
  * [Hadoop Deployment (YARN)](/deploy-dremio/other-options/yarn-hadoop) - Deploy Dremio on a Hadoop cluster using YARN.
  * [Dremio on Your Infrastructure](/deploy-dremio/other-options/standalone) - Deploy Dremio as a standalone cluster.

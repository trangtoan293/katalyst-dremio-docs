# LOCALTIME
Returns the current time for the system.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### LOCALTIME() → time[​](/reference/sql/sql-functions#localtime--time "Direct link to LOCALTIME\(\) → time")
**Examples**
LOCALTIME example

```
SELECT LOCALTIME()  
-- 05:07:01  

```

# SHA512
Computes the 512-bit SHA-2 hash value of a string.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### SHA512(_expression_ varchar) → varchar[​](/reference/sql/sql-functions#sha512expression-varchar--varchar "Direct link to sha512expression-varchar--varchar")
  * expression: The string to hash.


**Examples**
SHA512 example

```
SELECT SHA512('Dremio')  
-- f011241f4a038756cbffccc934ab67215e0ca5760d84423af09307c0e6d9991b24963680db6102466176bcea4d609b82ed2e  

```

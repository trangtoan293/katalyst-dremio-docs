# CHARACTER_LENGTH
Returns the length of an input string.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### CHARACTER_LENGTH(_expression_ varchar) → int32[​](/reference/sql/sql-functions#character_lengthexpression-varchar--int32 "Direct link to character_lengthexpression-varchar--int32")
  * expression: String expression to determine the length of.


**Examples**
CHARACTER_LENGTH example

```
SELECT CHARACTER_LENGTH('DREMIO')  
-- 6  

```

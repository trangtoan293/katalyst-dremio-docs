# OCTET_LENGTH
Returns the length of the string in bytes.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### OCTET_LENGTH(_input_ varchar) → int32[​](/reference/sql/sql-functions#octet_lengthinput-varchar--int32 "Direct link to octet_lengthinput-varchar--int32")
  * input: The string for which the length is returned.


**Examples**
OCTET_LENGTH example

```
SELECT OCTET_LENGTH('abc')  
-- 3  

```

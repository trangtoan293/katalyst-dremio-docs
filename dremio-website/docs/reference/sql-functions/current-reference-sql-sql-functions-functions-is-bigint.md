# IS_BIGINT
Returns TRUE if the input expression is an big integer value.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### IS_BIGINT(_expression_ any) → boolean[​](/reference/sql/sql-functions#is_bigintexpression-any--boolean "Direct link to is_bigintexpression-any--boolean")
  * expression: Input expression.


**Examples**
IS_BIGINT example

```
SELECT IS_BIGINT(column_name)  
-- True  

```

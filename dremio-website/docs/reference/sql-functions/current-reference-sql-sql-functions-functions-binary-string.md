# BINARY_STRING
Converts the input expression to a binary value.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### BINARY_STRING(_expression_ VARCHAR) → BINARY[​](/reference/sql/sql-functions#binary_stringexpression-varchar--binary "Direct link to binary_stringexpression-varchar--binary")
  * expression: Varchar expression to convert to binary


**Examples**
BINARY_STRING example

```
SELECT BINARY_STRING('DREMIO')  
-- RFJFTUlP  

```

BINARY_STRING example

```
SELECT BINARY_STRING('000')  
-- MDAw  

```

# IS_UTF8
Returns whether an expression is valid UTF-8
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### IS_UTF8(_in_ any) → boolean[​](/reference/sql/sql-functions#is_utf8in-any--boolean "Direct link to is_utf8in-any--boolean")
  * in: an expression


**Examples**
IS_UTF8 example

```
select is_utf8('hello')  
-- True  

```

# NOW
Returns the current timestamp (date and time) in UTC timezone.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### NOW() → timestamp[​](/reference/sql/sql-functions#now--timestamp "Direct link to NOW\(\) → timestamp")
**Examples**
NOW example

```
SELECT NOW()  
-- 2021-07-02 04:55:55.267000  

```

# File and Table Formats
Dremio data sources support differing data formats. See [Manage Sources](/data-sources) for supported formats for specific data sources. The following data formats are supported by multiple data sources:
  * File Formats
    * Delimited text files, such as comma-separated values (CSV)
    * Excel (XLSX) and XLS
    * JSON
    * [Parquet](/developer/data-formats/parquet)
    * ORC
Currently, ORC files can be read only from a metastore source, such as Hive or Glue. They cannot be read from an object store or a filesystem source.
  * Table Formats
    * [Apache Iceberg](/developer/data-formats/apache-iceberg)
    * [Delta Lake](/developer/data-formats/delta-lake)

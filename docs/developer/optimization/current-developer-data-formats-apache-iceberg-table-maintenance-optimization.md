# Table Maintenance and Optimization
Efficiently managing your tables helps ensure that your queries run quickly and reliably. Dremio provides three key methods for managing tables:
  * [**Table Cleanup with Vacuum**](/developer/data-formats/apache-iceberg/table-maintenance-optimization/table-cleanup-vacuum) – Use the `VACUUM CATALOG` command to remove expired snapshots and orphaned metadata files.
  * [**Optimize Tables**](/developer/data-formats/apache-iceberg/table-maintenance-optimization/optimizing) – Use the `OPTIMIZE TABLE` command to compact small files, split large files, or repartition data.
  * [**Automated Maintenance with the Open Catalog**](/developer/data-formats/apache-iceberg/table-maintenance-optimization/automated-maintenance) - Enable automated optimization and table cleanup for your cluster's built-in [Open Catalog](/data-sources/open-catalog).

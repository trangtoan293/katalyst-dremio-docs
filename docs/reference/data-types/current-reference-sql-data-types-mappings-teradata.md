# Teradata Data Types
The following table shows the mappings from Teradata to Dremio data types. If there are additional Teradata types not listed in the table, then those types are not supported in Dremio.  
| Teradata Database Type  | Dremio Type  |  
| --- | --- |  
| BIGINT  | BIGINT  |  
| BLOB  | VARBINARY  |  
| BYTE  | VARBINARY  |  
| BYTEINT  | INTEGER  |  
| CHAR  | VARCHAR  |  
| DATE  | DATE  |  
| DECIMAL  | DECIMAL  |  
| FLOAT  | DOUBLE  |  
| GRAPHIC  | VARCHAR  |  
| INTEGER  | INTEGER  |  
| LONG VARCHAR  | VARCHAR  |  
| LONG VARGRAPHIC  | VARCHAR  |  
| LONGVARCHAR  | VARCHAR  |  
| NUMBER  | DECIMAL  |  
| SMALLINT  | INTEGER  |  
| TIME WITH TIMEZONE  | TIME  |  
| TIME  | TIME  |  
| TIMESTAMP WITH TIMEZONE  | TIMESTAMP  |  
| TIMESTAMP  | TIMESTAMP  |  
| VARBYTE  | VARBINARY  |  
| VARCHAR  | VARCHAR  |  
| VARGRAPHIC  | VARCHAR  |

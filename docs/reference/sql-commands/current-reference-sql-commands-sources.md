# Source SQL Statements
Source statuses can be managed using SQL commands.
## Refreshing Source Status[​](/reference/sql/commands/sources#refreshing-source-status "Direct link to Refreshing Source Status")
This command will refresh source status for a given source:
Syntax

```
ALTER SOURCE <SOURCE_NAME> REFRESH STATUS  

```

# CREATE SPACE
Create a new space.
Syntax

```
CREATE SPACE <space_name>  

```

## Parameters[​](/reference/sql/commands/create-space#parameters "Direct link to Parameters")
`space_name` String
The name of the space that you are creating.
* * *
## Examples[​](/reference/sql/commands/create-space#examples "Direct link to Examples")
Create a space

```
CREATE SPACE my_space  

```

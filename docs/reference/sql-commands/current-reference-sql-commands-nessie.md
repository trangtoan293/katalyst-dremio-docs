# SQL Commands for Nessie
Using SQL commands, you can create, alter, show, and drop branches and tags; and also merge branches. For more information about working with branches and tags in Nessie, see [Nessie](/data-sources/lakehouse-catalogs/nessie).
A Nessie source needs to be configured for these commands to work. For information, see [Configuring Nessie as a Source](/data-sources/lakehouse-catalogs/nessie).
## Supported SQL commands​
  * [ALTER BRANCH](/reference/sql/commands/nessie/alter-branch)
  * [ALTER TAG](/reference/sql/commands/nessie/alter-tag)
  * [CREATE BRANCH](/reference/sql/commands/nessie/create-branch)
  * [CREATE FOLDER](/reference/sql/commands/nessie/create-folder)
  * [CREATE TAG](/reference/sql/commands/nessie/create-tag)
  * [DROP BRANCH](/reference/sql/commands/nessie/drop-branch)
  * [DROP TAG](/reference/sql/commands/nessie/drop-tag)
  * [MERGE BRANCH](/reference/sql/commands/nessie/merge-branch)
  * [SHOW BRANCHES](/reference/sql/commands/nessie/show-branches)
  * [SHOW TAGS](/reference/sql/commands/nessie/show-tags)
  * [VACUUM CATALOG](/reference/sql/commands/nessie/vacuum-catalog)

# COS
Computes the cosine of a value in radians.
## Syntax
### COS(_numeric_expression_ NUMERIC) → FLOAT[​](/reference/sql/sql-functions)
  * numeric_expression: The number in radians. This must be DOUBLE, INTEGER, or FLOAT.


**Examples**
COS example

```
SELECT COS(0)  
-- 1.0  

```

COS example

```
SELECT COS(1.0)  
-- 0.5403023058681398  

```

COS example

```
SELECT COS(-1)  
-- 0.5403023058681398  

```

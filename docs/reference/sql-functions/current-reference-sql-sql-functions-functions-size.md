# SIZE
Returns the number of entries in a map expression.
## Syntax
### SIZE(_input_ map) → int[​](/reference/sql/sql-functions)
  * input: A map expression for which to return the number of entries.


**Examples**
SIZE example

```
SELECT SIZE(properties)  
-- 3  
-- 2  

```

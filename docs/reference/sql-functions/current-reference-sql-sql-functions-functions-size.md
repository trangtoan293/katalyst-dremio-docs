# SIZE
Returns the number of entries in a map expression.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### SIZE(_input_ map) → int[​](/reference/sql/sql-functions#sizeinput-map--int "Direct link to sizeinput-map--int")
  * input: A map expression for which to return the number of entries.


**Examples**
SIZE example

```
SELECT SIZE(properties)  
-- 3  
-- 2  

```

# BITWISE_XOR
Returns the bitwise XOR of two operands.
## Syntax
### BITWISE_XOR(_op1_ NUMERIC, _op2_ NUMERIC) → NUMERIC[​](/reference/sql/sql-functions)
  * op1: First operand.
  * op2: Second operand.


**Examples**
BITWISE_XOR example

```
SELECT BITWISE_XOR(7, 4)  
-- 3  

```

BITWISE_XOR example

```
SELECT BITWISE_XOR(1, 2)  
-- 3  

```

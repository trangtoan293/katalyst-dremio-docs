# IS_VARCHAR
Returns TRUE if the input expression is a varchar value.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### IS_VARCHAR(_expression_ any) → boolean[​](/reference/sql/sql-functions#is_varcharexpression-any--boolean "Direct link to is_varcharexpression-any--boolean")
  * expression: Input expression.


**Examples**
IS_VARCHAR example

```
SELECT IS_VARCHAR(column_name)  
-- True  

```

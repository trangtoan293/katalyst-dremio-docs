# FROM_HEX
Returns a binary value for the given hexadecimal string.
## Syntax
### FROM_HEX(_in_ string) → binary[​](/reference/sql/sql-functions)
  * in: A hexadecimal string


**Examples**
FROM_HEX example

```
select from_hex('3fd98a3c')  
-- P9mKPA==  

```

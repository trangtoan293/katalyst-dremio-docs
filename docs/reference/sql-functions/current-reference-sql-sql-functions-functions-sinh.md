# SINH
Computes the hyperbolic sine of the input expression.
## Syntax
### SINH(_numeric_expression_ int32) → double[​](/reference/sql/sql-functions)
  * numeric_expression: Input expression.


**Examples**
SINH example

```
SELECT SINH(1)  
-- 1.1752011936438014  

```

### SINH(_numeric_expression_ float) → double[​](/reference/sql/sql-functions)
  * numeric_expression: Input expression.


**Examples**
SINH example

```
SELECT SINH(1.5)  
-- 2.1292794550948173  

```

### SINH(_numeric_expression_ double) → double[​](/reference/sql/sql-functions)
  * numeric_expression: Input expression.

SINH example

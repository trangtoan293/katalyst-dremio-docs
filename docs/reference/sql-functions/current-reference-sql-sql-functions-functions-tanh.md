# TANH
Computes the hyperbolic tangent of the input expression.
## Syntax
### TANH(_numeric_expression_ double) → double[​](/reference/sql/sql-functions)
  * numeric_expression: Input expression to calculate tanh for.


**Examples**
TANH example

```
SELECT TANH(1.5);  
-- 0.9051482536448664  

```

### TANH(_numeric_expression_ int64) → double[​](/reference/sql/sql-functions)
  * numeric_expression: Input expression to calculate tanh for.


**Examples**
TANH example

```
SELECT TANH(1);  
-- 0.7615941559557649  

```

# ISDATE
Returns `true` if the input expression can be cast to a date.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### ISDATE(_expression_ varchar) → boolean[​](/reference/sql/sql-functions#isdateexpression-varchar--boolean "Direct link to isdateexpression-varchar--boolean")
  * expression: Input expression.


**Examples**
ISDATE example

```
SELECT ISDATE('2021-04-22')  
-- True  

```

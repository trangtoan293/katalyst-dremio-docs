# ATAN
Computes the Arctangent (inverse Tangent) of a value.
## Syntax
### ATAN(_inputValue_ FLOAT) → FLOAT[​](/reference/sql/sql-functions)
  * inputValue: Floating-point input value, in the range (negative-infinity:positive-infinity)


**Examples**
ATAN example

```
SELECT ATAN(0)  
-- 0.0  

```

ATAN example

```
SELECT ATAN(1)  
-- 0.7853981633974483  

```

ATAN example

```
SELECT ATAN(-1)  
-- -0.7853981633974483  

```

ATAN example

```
SELECT ATAN(19564.7)  
-- 1.5707452143321894  

```

# IS_INT
Returns TRUE if the input expression is an integer value.
## Syntax
### IS_INT(_expression_ any) → boolean[​](/reference/sql/sql-functions)
  * expression: Input expression.


**Examples**
IS_INT example

```
SELECT IS_INT(column_name)  
-- True  

```

# BITWISE_AND
Returns the bitwise AND of two operands.
## Syntax
### BITWISE_AND(_op1_ NUMERIC, _op2_ NUMERIC) → NUMERIC[​](/reference/sql/sql-functions)
  * op1: First operand
  * op2: Second operand


**Examples**
BITWISE_AND example

```
SELECT BITWISE_AND(7, 4)  
-- 4  

```

BITWISE_AND example

```
SELECT BITWISE_AND(1, 2)  
-- 0  

```

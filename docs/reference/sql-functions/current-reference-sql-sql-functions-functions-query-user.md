# QUERY_USER
Returns the username of the user that is currently logged in to the system. This function cannot be used in a Reflection.
## Syntax
### QUERY_USER() → varchar[​](/reference/sql/sql-functions) → varchar")
**Examples**
QUERY_USER example

```
SELECT QUERY_USER()  
-- testuser@dremio.com  

```

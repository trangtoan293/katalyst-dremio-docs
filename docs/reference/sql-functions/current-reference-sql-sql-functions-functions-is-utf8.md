# IS_UTF8
Returns whether an expression is valid UTF-8
## Syntax
### IS_UTF8(_in_ any) → boolean[​](/reference/sql/sql-functions)
  * in: an expression


**Examples**
IS_UTF8 example

```
select is_utf8('hello')  
-- True  

```

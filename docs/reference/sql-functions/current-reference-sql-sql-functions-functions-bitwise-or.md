# BITWISE_OR
Returns the bitwise OR of two operands.
## Syntax
### BITWISE_OR(_op1_ NUMERIC, _op2_ NUMERIC) → NUMERIC[​](/reference/sql/sql-functions)
  * op1: First operand.
  * op2: Second operand.


**Examples**
BITWISE_OR example

```
SELECT BITWISE_OR(7, 4)  
-- 7  

```

BITWISE_OR example

```
SELECT BITWISE_OR(1, 2)  
-- 3  

```

# XOR
Returns the bitwise XOR of two integers.
## Syntax
### XOR(_expression1_ int32, _expression2_ int32) → int32[​](/reference/sql/sql-functions)
  * expression1: The integer input.
  * expression2: The integer input.


**Examples**
XOR example

```
SELECT XOR(1, 1)  
-- 0  

```

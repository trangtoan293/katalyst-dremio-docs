# XOR
Returns the bitwise XOR of two integers.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### XOR(_expression1_ int32, _expression2_ int32) → int32[​](/reference/sql/sql-functions#xorexpression1-int32-expression2-int32--int32 "Direct link to xorexpression1-int32-expression2-int32--int32")
  * expression1: The integer input.
  * expression2: The integer input.


**Examples**
XOR example

```
SELECT XOR(1, 1)  
-- 0  

```

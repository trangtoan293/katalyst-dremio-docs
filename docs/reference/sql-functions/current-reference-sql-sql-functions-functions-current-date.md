# CURRENT_DATE
Returns the current date of the system.
## Syntax
### CURRENT_DATE() → date[​](/reference/sql/sql-functions) → date")
**Examples**
CURRENT_DATE example

```
SELECT CURRENT_DATE()  
-- 2021-07-02  

```

CURRENT_DATE example

```
SELECT CURRENT_DATE  
-- 2021-07-02  

```

## Usage Notes[​](/reference/sql/sql-functions)
This function can be called without the parentheses.

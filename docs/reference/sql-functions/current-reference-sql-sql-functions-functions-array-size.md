# ARRAY_SIZE
Returns the size of the input array.
## Syntax
### ARRAY_SIZE(_arr_ LIST) → numeric[​](/reference/sql/sql-functions)
  * arr: The source array.


**Examples**
ARRAY_SIZE example

```
SELECT ARRAY_SIZE(ARRAY[1, 4, 5])  
-- 3  

```

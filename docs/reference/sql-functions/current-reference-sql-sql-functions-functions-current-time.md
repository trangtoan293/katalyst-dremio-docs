# CURRENT_TIME
Returns the current time for the system.
## Syntax
### CURRENT_TIME() → TIME[​](/reference/sql/sql-functions) → TIME")
**Examples**
CURRENT_TIME example

```
SELECT CURRENT_TIME()  
-- 06:04:31  

```

CURRENT_TIME example

```
SELECT CURRENT_TIME  
-- 06:04:31  

```

## Usage Notes[​](/reference/sql/sql-functions)
This function can be called without the parentheses.

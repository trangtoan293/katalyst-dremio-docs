# REVERSE
Reverses the order of characters in a string.
## Syntax
### REVERSE(_expression_ varchar) → varchar[​](/reference/sql/sql-functions)
  * expression: The string to reverse.


**Examples**
REVERSE example

```
SELECT REVERSE('Hello, world!');  
-- !dlrow ,olleH  

```

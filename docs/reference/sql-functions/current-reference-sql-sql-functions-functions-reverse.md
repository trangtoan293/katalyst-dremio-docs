# REVERSE
Reverses the order of characters in a string.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### REVERSE(_expression_ varchar) → varchar[​](/reference/sql/sql-functions#reverseexpression-varchar--varchar "Direct link to reverseexpression-varchar--varchar")
  * expression: The string to reverse.


**Examples**
REVERSE example

```
SELECT REVERSE('Hello, world!');  
-- !dlrow ,olleH  

```

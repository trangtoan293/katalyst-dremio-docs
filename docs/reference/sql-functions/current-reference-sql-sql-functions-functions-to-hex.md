# TO_HEX
Returns a hexadecimal string for the given binary value.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### TO_HEX(_in_ binary) → string[​](/reference/sql/sql-functions#to_hexin-binary--string "Direct link to to_hexin-binary--string")
  * in: A binary value


**Examples**
TO_HEX example

```
select to_hex(binary_string('hello'))  
-- 68656C6C6F  

```

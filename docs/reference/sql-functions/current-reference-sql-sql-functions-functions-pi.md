# PI
Returns the value of pi, which is approximately 3.14592654.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### PI() → double[​](/reference/sql/sql-functions#pi--double "Direct link to PI\(\) → double")
**Examples**
PI example

```
SELECT PI()  
-- 3.141592653589793  

```

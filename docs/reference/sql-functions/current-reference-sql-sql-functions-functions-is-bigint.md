# IS_BIGINT
Returns TRUE if the input expression is an big integer value.
## Syntax
### IS_BIGINT(_expression_ any) → boolean[​](/reference/sql/sql-functions)
  * expression: Input expression.


**Examples**
IS_BIGINT example

```
SELECT IS_BIGINT(column_name)  
-- True  

```

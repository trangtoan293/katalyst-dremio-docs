# RADIANS
Convert a value in degrees to radians.
## Syntax
### RADIANS(_x_ number) → float[​](/reference/sql/sql-functions)
  * x: The number in degrees


**Examples**
RADIANS example

```
select radians(45)  
-- 0.7853981633974483  

```

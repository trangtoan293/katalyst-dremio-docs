# RADIANS
Convert a value in degrees to radians.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### RADIANS(_x_ number) → float[​](/reference/sql/sql-functions#radiansx-number--float "Direct link to radiansx-number--float")
  * x: The number in degrees


**Examples**
RADIANS example

```
select radians(45)  
-- 0.7853981633974483  

```

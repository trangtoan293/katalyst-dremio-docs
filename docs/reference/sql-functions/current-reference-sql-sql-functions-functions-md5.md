# MD5
Computes the MD5 hash value of a string.
## Syntax
### MD5(_expression_ varchar) → varchar[​](/reference/sql/sql-functions)
  * expression: The string to hash.


**Examples**
MD5 example

```
SELECT MD5('Dremio')  
-- 288e0e9ab8b8ac8737afefecf16f61fd  

```

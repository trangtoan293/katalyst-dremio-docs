# BASE64
Returns the Base64 encoding of a binary string.
## Syntax
### BASE64(_expression_ varbinary) → varchar[​](/reference/sql/sql-functions)
  * expression: The string to encode.


**Examples**
BASE64 example

```
SELECT BASE64('Dremio')  
-- RHJlbWlv  

```

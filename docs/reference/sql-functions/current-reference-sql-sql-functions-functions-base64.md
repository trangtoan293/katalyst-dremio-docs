# BASE64
Returns the Base64 encoding of a binary string.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### BASE64(_expression_ varbinary) → varchar[​](/reference/sql/sql-functions#base64expression-varbinary--varchar "Direct link to base64expression-varbinary--varchar")
  * expression: The string to encode.


**Examples**
BASE64 example

```
SELECT BASE64('Dremio')  
-- RHJlbWlv  

```

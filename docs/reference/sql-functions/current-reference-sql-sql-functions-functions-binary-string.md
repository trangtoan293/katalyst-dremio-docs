# BINARY_STRING
Converts the input expression to a binary value.
## Syntax
### BINARY_STRING(_expression_ VARCHAR) → BINARY[​](/reference/sql/sql-functions)
  * expression: Varchar expression to convert to binary


**Examples**
BINARY_STRING example

```
SELECT BINARY_STRING('DREMIO')  
-- RFJFTUlP  

```

BINARY_STRING example

```
SELECT BINARY_STRING('000')  
-- MDAw  

```

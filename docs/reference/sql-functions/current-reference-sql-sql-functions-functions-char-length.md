# CHAR_LENGTH
Returns the character length of the input string.
## Syntax[​](/reference/sql/sql-functions#syntax "Direct link to Syntax")
### CHAR_LENGTH(_expression_ STRING) → INTEGER[​](/reference/sql/sql-functions#char_lengthexpression-string--integer "Direct link to char_lengthexpression-string--integer")
  * expression: The expression (VARCHAR) to determine character length for


**Examples**
CHAR_LENGTH example

```
SELECT CHAR_LENGTH('get the char length')  
-- 19  

```

CHAR_LENGTH example

```
SELECT CHAR_LENGTH('DREMIO')  
-- 6  

```

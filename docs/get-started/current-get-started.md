# Get Started with Dremio
Welcome to Dremio, a data lakehouse platform that facilitates high-performance, self-service analytics on large datasets.
To get started with Dremio, you have two options for a hands-on experience:
  * [Enterprise Edition Free Trial](https://docs.dremio.com/current/get-started/kubernetes-trial/) (Recommended) - Deploy and explore Dremio on Kubernetes using the Enterprise Edition free trial with all features unlocked.
  * [Community Edition on Docker](https://docs.dremio.com/current/get-started/docker/) - Deploy and explore Dremio on Docker using the Community Edition with a limited set of features. This option provides a local deployment with a single node, which is suggested for testing and evaluation purposes.


Choose the option that best fits your needs. To learn about the differences between the two editions, see [Dremio Editions](https://docs.dremio.com/editions).

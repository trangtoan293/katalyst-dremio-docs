# Uploading Profile Data
This topic describes how to upload job profile data files in Dremio as a dataset, so you can use SQL for analyzing the profile data. You can either upload your **profile_attempt_0.json** file or use the sample **profile_attempt_0.json** file below. For more information, see [Uploading files](https://docs.dremio.com/current/data-sources/file-upload/).
To upload the profile data:
  1. Download your **profile_attempt_0.json** file or the sample **profile_attempt_0.json** file.


  * For your **profile_attempt_0.json** file, go to [Download Profile](https://docs.dremio.com/current/admin/monitoring/jobs/overview#3-download-profile).
  * For the sample **profile_attempt_0.json** file, use the [profile_attempt_0.json](https://docs.dremio.com/current/admin/monitoring/jobs/view-profiles/uploading-profile-data#sample-profile_attempt_0) file below.


  1. Click the Datasets ![This is the icon that represents the Datasets UI.](https://docs.dremio.com/images/datasets-icon.png) (grid) icon in the side navigation bar.
  2. Click the Add ![This is the Add icon.](https://docs.dremio.com/images/upload-icon.png) (plus) icon in the top-right corner and select **Upload File**.
  3. Drop a local file into the popup or browse to add the file.
  4. Click **Next** and then click **Save**.


The downloaded **profile_attempt_0.json** file is now uploaded and in Dremio as a dataset.

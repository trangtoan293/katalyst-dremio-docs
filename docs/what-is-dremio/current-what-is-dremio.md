# What is Dremio?
Dremio enables self-service analytics for data teams. Data analysts can explore and visualize data with sub-second query response times, and data engineers can ingest and transform data directly in the data lake with full support for DML operations. In addition, analysts can join data in the lake with data in external databases, so they don't have to move data into object storage to derive value from that data. Dremio's open lakehouse platform, based on community-driven standards like Apache Iceberg and Apache Arrow, enables organizations to use best-in-class processing engines and eliminates vendor lock-in.

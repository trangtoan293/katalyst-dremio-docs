# AWS Edition (Deprecated)
The AWS Edition (AWSE) is no longer available in Dremio 26.0+, but it is **still supported in version 25.x**. For end-of-life dates, see [Dremio Supported Software](https://www.dremio.com/legal/dremio-supported-software/). For more information on AWSE, refer to the following documentation for version 25.x:
  * [AWS Edition](/25.x/get-started/cluster-deployments/deployment-models/amazon-deployments/aws) - The overview of AWSE for Dremio 25.x.
  * [Migrate AWSE to Kubernetes](/25.x/admin/cli/awse-migration) - The procedure in Dremio 25.x to migrate from AWS Edition to Dremio's Enterprise Edition on Kubernetes before upgrading to Dremio 26.0+.


Dremio will help migrate your Dremio AWSE Enterprise environment to Dremio Enterprise Version 26 on Kubernetes or Dremio Cloud. For further details, please contact your Dremio account executive or Dremio Support.
